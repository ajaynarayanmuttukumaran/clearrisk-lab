# Risk Methodology (Draft)

This project explores one guiding question:

When can a CCP margin and default waterfall setup appear robust under standard assumptions but become fragile under tail risk, wrong-way risk, liquidity close-out costs, concentration, and assessment pressure?

Method blocks:
- Market shocks (Gaussian baseline plus stressed and tail variants).
- Margin regime comparison (VaR/ES/stressed variants).
- Default waterfall allocation by explicit layers.
- Cover-2 adequacy and contagion-oriented diagnostics.
