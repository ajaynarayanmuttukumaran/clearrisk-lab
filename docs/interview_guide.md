# Interview Guide (Draft)

## 30-Second Pitch
ClearRisk Lab is an educational CCP stress-testing simulator that examines margin design, tail-risk assumptions, and default waterfall resilience under stressed clearing member behavior.

## 2-Minute Technical Summary
- Synthetic multi-regime scenario engine.
- Modular margin and default waterfall components.
- Explicit analysis of procyclicality, Cover-2 adequacy, liquidity close-out costs, and assessment contagion.

## Integrity Statement
The model is intentionally transparent and educational, not a production CCP model.
